CSS (Cascading Style Sheets) é uma linguagem usada para estilizar páginas web. Com CSS, você pode alterar cores, fontes e layouts, separando o conteúdo HTML da apresentação visual. Isso torna o design de sites mais organizado e facilita a manutenção.

Flexbox e Grid são ferramentas CSS para criar layouts modernos. Flexbox alinha itens em uma linha ou coluna, enquanto o Grid organiza elementos em uma grade bidimensional. Ambos ajudam a construir designs responsivos e adaptáveis a diferentes tamanhos de tela.
